# OssoSaurus: Desenterrando o Passado

## Controlos

**Andar**: Setas\
**Escavar**: Barra de espaço

## Itens

**Espinha de peixe:** 5 pts\
**Osso:** 10 pts\
**Moedas:** 20 pts\
**Cálice:** 30 pts\
**Coroa:** 40 pts\
**Diamante:** 50 pts\

**Poção:** Cura o efeito causado pela caveira
